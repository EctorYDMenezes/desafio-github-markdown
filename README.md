# 🚀 Desafio GitHub Markdown

Este repositório foi criado para a realização do desafio **GitHub Markdown** da DIO.

## 📚 Objetivo

Praticar os principais conceitos de:

- Git
- GitHub
- Markdown

## 🛠️ Tecnologias

- Git
- GitHub
- Markdown

## 📖 O que foi praticado

- Criação de repositório
- Edição de arquivos Markdown
- Commits
- Push para o GitHub
- Fork
- Pull Request

## 🚀 Como realizar este desafio

### 1. Faça um Fork do repositório original

Clique em **Fork** no repositório do desafio para criar uma cópia na sua conta.

### 2. Clone o seu repositório

> **Troque `SEU_USUARIO` pelo seu usuário do GitHub.**

```bash
git clone https://github.com/SEU_USUARIO/desafio-github-markdown.git
```

Entre na pasta do projeto:

```bash
cd desafio-github-markdown
```

### 3. Edite o README

Faça as alterações solicitadas pelo desafio utilizando Markdown.

### 4. Verifique as alterações

```bash
git status
```

### 5. Adicione os arquivos

```bash
git add .
```

### 6. Faça um commit

```bash
git commit -m "docs: atualiza README do desafio"
```

### 7. Envie para o GitHub

```bash
git push origin main
```

### 8. (Opcional) Abra um Pull Request

Caso o desafio solicite, abra um Pull Request do seu fork para o repositório original.
